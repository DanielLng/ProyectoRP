Interpretación

Operación Frases Expresión algebraica
+
( Suma )		 1. Un número más 5.
2. Sumar 5 a un número.
3. Suma de un número y 5.
4. 5 más que un número.
5. Un número aumentado en 5. 		n + 5
-
( Resta )		 1. Un número menos de 11.
2. Restar 11 a un número.
3. Diferencia entre un número y 11
4. 11 menos que un número
5. Un número disminuido en 11. 		x - 11
*
( Producto ) 		1. 3 veces un número.
2. 3 veces multiplicado por un número.
3. El producto de 3 y un número. 		3c
/
( División )		 1. Un número divido entre 7.
2. La séptima parte de un número.
3. El cociente de un número y 7. 		b / 7
